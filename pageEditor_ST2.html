Splash Screen

HTML Structure

A single full-screen <section> holds the title, hook, group members, CSS loader and countdown. A <meta http-equiv="refresh"> tag redirects to the Home Page after 4 seconds.

CSS & JavaScript Techniques

The loader is a pure-CSS spinning ring built with @keyframes, and a slow background zoom sets a cinematic tone. JavaScript drives the visible countdown with setInterval, and the Skip button clears the timer and links straight to Home.

Action Impact Simulator

HTML Structure

Six action <div> cards each carry a data-points value, plus a live feedback area with aria-live.

JavaScript Techniques

Clicking a card toggles a selected class; JS sums the points, decides Low/Medium/High level using three thresholds, updates the message, and swaps the page background image so the land visibly heals. Five embedded CSS rules style the cards and selected state.

Content Page 2

Covers Wildlife & Biodiversity with four sections, four images, in-page anchor navigation and a CSS-only Go-to-Top button.

Challenges & Lessons Learned

Getting the background image to change smoothly without hurting text readability was tricky. I added a translucent panel behind the content and a CSS transition on the background. I learned how JavaScript state (the running total) can drive visual storytelling.

Accessibility Considerations

Cards are keyboard-focusable and respond to Enter/Space, the feedback area uses aria-live="polite", all images have alt text, and the loader has an aria-label. Contrast stays readable over every background image, and the splash animation respects prefers-reduced-motion.

Compliance with JANET Regulations

Content is original, educational and non-offensive; images are royalty-free and credited. The pages promote lawful, responsible messaging in line with JANET acceptable-use principles.
